# Fatality video prompts — 8s multi-reference Seedream/Seedance test

Experiment (2026-07-03): replace the 4 still cutscene panels per fatality with one
8-second generated video (+ generated audio fx where the model supports it).
Reference image passed to the model: the character's **canonical sheet only**
(same paths as `tools/gen-fatality.mjs` `canonical:` fields). The victim is the
generic burnt husk, described in text — no second reference.

Shared constraints baked into every prompt: 16:9, painted cel-shaded anime
cutscene style (Arc System Works cinematic super aesthetic), speed-line/energy
background with no scenery, no on-screen text or watermarks (Gene's error
dialog is the one deliberate exception), single continuous shot with timed
beats so the 4-panel arc survives compression into 8 seconds.
