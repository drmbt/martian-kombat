# Martian Kombat — locked art style (approved 2026-07-01)

Every character-frame prompt MUST include this base verbatim. Do not fork the
style per character; consistency across the roster is the whole point.

## Character style base

> Art style: hand-painted cel-shaded 2D anime fighter (modern Capcom / Arc
> System Works aesthetic). Bold clean line art, painterly cel shading,
> confident silhouette, slightly heroic proportions while keeping the person
> recognizable.

## Frame rules (append to every sprite-frame prompt)

> Same character as the reference image — identical face, hair, outfit,
> colors, proportions and art style. Full body visible head to toe, feet
> resting on an invisible ground line just above the bottom edge, same scale
> and camera distance as the reference image, character centered, facing
> right. Background: solid flat chroma-key green (#00B140), completely
> uniform, no cast shadows, no floor, no text, no watermark, no border.

## Stage style base

> Painted 2D fighting game stage background, cinematic wide shot, rich color
> and light, layered composition with parallax-friendly depth. The bottom
> quarter is a clean, flat, unobstructed ground plane where two fighters will
> stand. No text, no watermark, no UI.

Approved MVP stage: `salton-shoreline` (gpt-image-2). Approved character
canon: `assets/raw/style-tests/char-*-b-painted.png` are the canonical
reference sheets for Vincent and Yulia.
